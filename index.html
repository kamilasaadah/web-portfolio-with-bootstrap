MY PORTFOLIO

Recent Projects

Here are some of my recent projects that showcase my skills and expertise in web development, UI/UX design, and more.

DecentGrowth
Software Requirements

DecentGrowth is a web-based digital platform designed to help individuals from disadvantaged backgrounds find and access decent employment opportunities. The platform connects job seekers with potential employers through skills-based matching, interactive location maps, and a secure online payment system. This project focuses on developing an SRS (Software Requirements Specification) document.

MediStore
Laravel

MediStore is a web-based digital platform that serves as an information and drug ordering system for the general public. It enables pharmacies to manage their drug stock more efficiently while providing convenience for users to order medicine online. The platform is built using PHP with the Laravel framework and MySQL for the backend.

RailKu
Prototype UI/UX

RailKu is a train ticket booking application designed to make it easier and more convenient for users to book intercity train tickets for long-distance travel between major cities.

BansosFix
Project Management

BansosFix is a digital solution designed to ensure the validation of poverty data in the 10 poorest regions of East Java is more transparent and accurate. This system allows IT-based survey teams to verify the validity of aid recipient data reported by local officials.

Panenia
Frontend Development

Panenia is an e-commerce website for agricultural products such as vegetables, fruits, spices, and other essentials. It includes features like online ordering, payment processing, shopping cart, purchase history, related articles, and an admin dashboard for managing sales. The project is built using HTML, CSS, and JavaScript.

Sweetnest Bakery
Frontend Development

A UI/UX landing page design project built using HTML and CSS, applying structured layout sections for a clean and visually appealing interface.

Pastry Paradise
Mobile App Programming

A full-stack mobile application project about pastry recipes, developed using Dart and the Flutter framework with PocketBase as the backend.

Tower Defense
Artificial Intelligence

A project that implements artificial intelligence in a tower defense game using Python, applying the Mamdani fuzzy logic concept.
